# MALL-VUE

> 这是一个基于VUE + VUEX + iView做的一个电商网站前端项目, 欢迎fork或star

## Build Setup

``` bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 打包
npm run build
```
